Types of lists

There are three types of lists in HTML:

  • Ordered lists - used when the order of items is important. They are created using the <ol> tag.
  • Unordered lists - used when the order of items is not important. They are created using the <ul> tag.
  • Description lists - used to define terms and their descriptions. They are created using the <dl> tag.
